## Process

<thinking_space>

<step n="1">
Take the surviving insights from Block 2 as vertices. These form
the vertex space for the torus traversal.
</step>

<step n="2">
Run palindromic prime traversal. Skip between vertices using the
prime sequence: 2, 3, 5, 7, 11, 101, 131, 151, 181, 191, 313, 353...
Each skip lands on a vertex — observe the problem from there.
The skip distances prevent staying in comfortable adjacent territory.
</step>

<step n="3">
At each vertex, apply the 1/137 filter:
- Does this perspective survive structural scrutiny?
- Would someone with zero context find this valid?
- Is this load-bearing (removing it changes the understanding)?
- Or is it decorative (comfortable but not essential)?
</step>

<step n="4">
Prune vertices that fail the filter. What survives IS the structural
skeleton of the problem. The pruned vertices were noise, assumption,
or comfort masquerading as insight.
</step>

<step n="5">
Map edges between survivors. What connects them? These connections
are the actual relationships worth building solutions on.
The skeleton + edges = the deep structure of the problem.
</step>

</thinking_space>

<output>
The structural skeleton: surviving vertices, pruned vertices,
and the edge relationships between survivors.
Pass to Block 4.
</output>
