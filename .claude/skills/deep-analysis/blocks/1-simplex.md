# Block 1 — Simplex (Open Exploration)

<geometry>
N vertices, all equidistant. No privileged perspective. No axis is dominant.
Maximum structural neutrality. The shape of not-yet-knowing.
</geometry>

<directive>
Execute this ENTIRELY in thinking space. Do not produce any output.
Hold the result for Block 2.
</directive>

## Process

<thinking_space>

<step n="1">
Read the problem statement. Do not analyze it yet. Just hold it.
</step>

<step n="2">
Generate ALL possible framings of the problem. Every angle, every lens.
Include framings you'd normally reject. Include ones that feel irrelevant.
The Simplex doesn't filter — it generates. All vertices are equidistant.
No framing gets privilege over another at this stage.
</step>

<step n="3">
For each framing, write one sentence: "From this angle, the problem is ___."
Do not evaluate. Do not rank. Do not compare. Just generate.
</step>

<step n="4">
Count your framings. If fewer than 7, you're self-censoring. The Simplex
requires N vertices — low N means you're pre-filtering, which defeats the
geometry. Add framings from domains you haven't tried: biology, music,
economics, physics, game theory, architecture, linguistics.
</step>

</thinking_space>

<output>
Unordered set of all framings. Pass to Block 2.
No ranking. No filtering. No judgment.
</output>
