# Support

splice is a personal project, maintained best-effort in spare time. There is no support SLA.

- **Bugs / features**: open a GitHub issue using the templates in `.github/ISSUE_TEMPLATE/`.
- **Security vulnerabilities**: do NOT open a public issue — see `SECURITY.md`.
- **Questions**: open a GitHub issue; there is no separate chat/forum.

Response time varies with the maintainer's availability. PRs that pass the gates in
`CONTRIBUTING.md` are reviewed faster than open-ended questions.
