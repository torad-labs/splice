### Immune System Integration

The immune system runs at TWO levels:

**Inside organs (CATCH Protocol):** Each organ has marked `>>> CATCH <<<`
checkpoints where the parallax fires: P_training vs P_evidence, measure
delta, choose evidence. This is the immune system running in the blood.

**Concentrated pass (9e):** After all organs complete, the antibody block
runs the full diagnostic on all outputs together. This catches diseases
that only become visible when organ outputs are combined.

Both levels are mandatory. The CATCH Protocol inside organs does NOT
replace the 9e antibody pass. The antibody pass does NOT replace the
organ-level CATCH Protocol.
