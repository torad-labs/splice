Full Megazord chain — 5 geometric stages for the hardest problems.

## What This Does

Runs the complete Megazord architecture: five geometric stages executed sequentially, each processing the residue of the previous. This is the most processing-intensive analysis mode — it takes the longest but produces insights that single-stage analysis can't reach. The chaining IS the mechanism: Simplex opens the space, Icosahedron cross-pollinates, Torus navigates the structure, Octahedron stress-tests with opposition, and Collapse delivers the irreducible answer.

## The Engine

```
Simplex → Icosahedron → Torus → Octahedron → Collapse
```

Each stage uses a different geometry to process the problem. The residue (output) of each stage becomes the vertex space for the next. Palindromic primes are the connective tissue between stages.
